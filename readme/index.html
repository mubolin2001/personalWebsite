Why CS?

很简单，我不知道自己想干什么，并且这个专业我学着还挺顺手的，并且有一定的兴趣。但有兴趣不代表我喜欢这个专业，因为这个专业大部分人相对于其他专业来说缺少那种骨子里的主见，我不太喜欢。（可能以后会往商科方向转？）

Hobbies？

  • Texas Hold’em：不是我喜欢赌博，而是因为这个游戏是一个与人斗的游戏，而且蕴含着很多资本的底层逻辑在里面（可能这是很多顶尖高手都是企业家的原因）。

  • 篮球：比较大众化吧可能，并且有点身体天赋在里面，好玩爱玩。

  • DOTA2：现在我唯一喜欢的MOBA游戏，也可能是我未来唯一的喜欢。虽然上手难度高，普及率小，但只要上手之后完全碾压其他MOBA游戏（玩过DOTA的应该可以理解）。

  • 🍺：好喝爱喝。

What kind of person?

  • 摆烂人：个人感觉自己挺活在当下的吧，可能会有短期目标但不会有长期的人生规划，想干点啥干点啥。（P以95%压倒性优势胜出）。

  • 喜欢第一印象：我觉着我对人的第一印象90%的时候都是准确的，我第一次见这个人喜欢这个人，我可能会一直喜欢这个人，反之亦然。

  • 喜欢表达但不喜欢总表达：我喜欢和想听我说话的人说话，而不是所有人。这和你理不理解，赞不赞同我的话没关系，只要喜欢听就好了。同时，即使我可能不赞同你说的话，但我喜欢你这个人，我也会听你说话 （和正确与否无关）。

  • 灰色世界：很喜欢一个朋友说的话，世界不是非黑即白的。这就是为什么我不喜欢探究对错，因为可能大部分事情都没有对错。

  • 现实主义者：人可以有理想， 但不能一直有理想。虽然获得大成功的人大多都是理想主义者，但大部分现实主义者生活会好一点。（但我很喜欢我的朋友或女朋友是理想主义者）。

  • 感觉至上：听起来感觉和现实主义很矛盾，实则不然。我喜欢用感觉来分析实际事件，并得出自己的结论。就比如说有些事情是很令人惊讶的，但是假如说这个令人惊讶的事是被我感觉是令人惊讶的人干出来的，我就觉得很正常了。

有可能会觉得我很矛盾：一个学CS的，臭玩电脑的人还在这谈感觉，笑死了。但我觉得并不冲突：逻辑和理性在工作中是很有必要的，但感觉在生活中会占主导地位。可能我平衡的也很烂，但我也会朝这个方向努力。

暂时这么多先，后续想到了可能会补充。

也不想过多介绍自己，估计能看到这篇blog基本都会认识我。我会在这个网站更新自己的想法，分享一些有趣的人，和对某些话题的看法。由于才疏学浅，我的文字不会很华丽，但每一字句都是我当下所知所感。不希望所有读过的人都能理解我的感受，只要读过之后会觉得：“哦居然还有人这么想。” 这就够了。
